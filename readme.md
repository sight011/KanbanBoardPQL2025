## Running

1. designed for [node.js lts v20](https://nodejs.org/en) and above

### Frontend

1. cd into ./frontend
1. npm install
1. npm run dev
1. open http://localhost:5173

### Backend

1. cd into ./backend
1. npm i
1. npm start
1. open http://localhost:4000
