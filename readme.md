## Running
(Designed for [node.js lts v20](https://nodejs.org/en) and above)


**On a Mac (for example)**

cd ~/Documents/New_Projects_2025/


git clone https://github.com/sight011/KanbanBoardPQL2025.git


### Frontend

1. cd KanbanBoardPQL2025/frontend
2. npm install
3. npm run dev
4. open http://localhost:5173

### Backend
(Open a new Terminal Window)

cd ~/Documents/New_Projects_2025/KanbanBoardPQL2025/backend

1. npm i
2. npm start
3. open http://localhost:4000

You are done! ü•≥
